Описание api сервиса

Список методов:

CURRENT_URL/find, где CURRENT_URL - тукещий url

Пареметры метода find

  • limit - Количество выбираемых данных
  • skip - Количество пропускаемых данных
  • tp - Тип звонка (H, I, J)
  • phone - номер телефона, например 98909556
  • stantion - Алиас тел.станции
  • called - Телефон
  • start и end - Интервал дат телефонного разговора, формат (год-месяц-день), например 2018-12-17 - 17 декабря 2018 года
    Вводится всегда парно

Пример запроса:

app_url/find?tp=H&start=2018-12-01&2018-12-31 - Выбрать тел. записи с указанными датами
app_url/find?phone=98909 - Выбрать все звонки, у которых телефон начинается на 98909
app_url/find?limit=5 - Выбрать 5 последних записей
app_url/find?limit=5&skip=10 - Выбрать 5 последних записей, пропустить 10 (что то вроде постраничной навигации, при таком запросе мы бы были на 3й странице)
